# xwebshell 1.0.0

`xwebshell 1.0.0` は、Kali LinuxのWeb Shellテンプレートを確認・出力するCLIです。

## 追加

- `xwebshell ls` を追加し、利用可能・不足・未登録のテンプレートを安定したID付きで一覧表示します。
- `xwebshell show <ID>` を追加し、選択したテンプレートのパスと内容を表示します。
- `xwebshell export <ID>` を追加し、選択したテンプレートを既存ファイルを上書きせずカレントディレクトリへ出力します。
- 複数ファイル構成のテンプレートはディレクトリ構成を維持して出力します。
- PerlおよびPHPリバースシェルの接続先IP・ポートを対話的に設定できます。
- 利用可能な場合は、非ループバックIPv4アドレスをCallback IPへ自動設定します。
- Magento管理者テンプレートの設定を対話的に行えます。
- Bash/Zsh補完を追加し、ZshではテンプレートIDと説明も表示します。
- `-h`/`--help` と `-V`/`--version` を追加します。
- Kali Linuxの`webshells`および`seclists`パッケージに依存します。
