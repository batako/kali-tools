# xwebshell 1.0.0

`xwebshell 1.0.0` provides a CLI for inspecting and exporting Kali web shell templates.

## Added

- Added `xwebshell ls` to list available, missing, and unregistered templates with stable IDs.
- Added `xwebshell show <ID>` to display template paths and contents.
- Added `xwebshell export <ID>` to export a selected template to the current directory without overwriting existing files.
- Preserve directory structure when exporting multi-file templates.
- Added interactive configuration for Perl and PHP reverse shell callback IP and port values.
- Automatically use a non-loopback IPv4 address for the callback IP when one is available.
- Added interactive configuration for Magento administrator templates.
- Added Bash and Zsh completion, including template IDs and descriptions in Zsh.
- Added `-h`/`--help` and `-V`/`--version`.
- Depend on the Kali `webshells` and `seclists` packages.
