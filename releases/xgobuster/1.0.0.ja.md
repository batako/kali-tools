# xgobuster 1.0.0

`xgobuster 1.0.0` は、現在のctxターゲットに対してGobusterを実行し、Web探索結果をctxへ保存します。

## 機能

- `xgobuster` CLIと短縮コマンド`xgo`を提供します。
- ctxのWebサービス情報から対象URLを決定し、HTTPとHTTPSに対応します。
- KaliのワードリストディレクトリからWebコンテンツ用ワードリストを自動選択します。
- `web-quick`、`web-standard`、`web-deep`の検索プロファイルに対応します。
- PHP、WordPress、ASP.NET、Java、Node.js、静的サイト向けのファイル検索プリセットに対応します。
- `-x`または`--extensions`でGobusterの拡張子を明示できます。
- 拡張子なしのパスはディレクトリ検索と検索履歴を共有し、拡張子付きのリクエストは個別に管理します。
- 設定したリクエスト数の範囲内で`--next`による検索継続に対応し、`--force`で明示的に再実行できます。
- `--status`でワードリストと検索状況を確認できます。
- `-w`または`--wordlist`でワードリストを明示できます。
- Gobusterのコマンド出力、状態、終了コード、発見したパスをctxへ保存します。
- `xgobuster`と`xgo`向けのBash・Zsh補完を提供します。

## 使い方

```sh
xgo
xgo --next
xgo --preset wordpress
xgo -x php,html,js
xgo --status
xgo --status --preset wordpress
```

`xgobuster` の実行には `ctx`、`gobuster`、Kaliの`wordlists`パッケージが必要です。

## 検索状態

Gobusterは`-x`指定時も拡張子なしのベースパスを確認するため、拡張子なしのパスはディレクトリ検索と共有します。一方、拡張子付きのリクエストは独立して管理するため、ディレクトリ検索だけでWordPressなどのファイル検索が完了扱いになることはありません。
