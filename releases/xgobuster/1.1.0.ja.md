# xgobuster 1.1.0

`xgobuster 1.1.0` は、収集したWeb探索結果の表示を改善します。

## 追加

- `xgo --sitemap` と `xgobuster --sitemap` を追加し、重複排除した探索結果をoriginごとに表示します。
- URLを自動決定する際、`xhost`で手動登録したホスト名を対象IPより優先して利用します。`-u`と`--url`による上書きも引き続き利用できます。
- 複数のホスト名が登録されている場合に備え、登録ホストを指定する`--host`と対象IPを強制する`--ip`を追加します。
- 期限切れや信頼されていないTLS証明書を使う検証環境向けに、`-k`と`--no-tls-validation`を追加します。
- GobusterのリクエストへCookieを付与する`-c`と`--cookies`を追加します。
- 共通のワイルドカードレスポンスをサイズで除外する`--exclude-length`を追加します。
- `--url`で指定したベースパスを含め、探索結果をURLルート基準のpathとして保存します。
- サイトマップ表示のHTTPステータスコードを、端末出力時に色分けします。

## 変更

- リダイレクト出力や非端末出力ではANSIカラーを無効にします。
- `NO_COLOR` が設定されている場合もカラーを無効にします。
