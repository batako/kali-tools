# xgobuster 1.1.0

`xgobuster 1.1.0` improves the collected web-discovery view.

## Added

- Added `xgo --sitemap` and `xgobuster --sitemap` to display deduplicated discoveries grouped by origin.
- Automatically uses a manually registered `xhost` hostname when deriving the target URL, with `-u` and `--url` remaining available as overrides.
- Added `--host` for selecting a registered hostname and `--ip` for forcing the target IP when multiple hostnames are registered.
- Added `-k` and `--no-tls-validation` for test targets with expired or otherwise untrusted TLS certificates.
- Added `-c` and `--cookies` for sending cookies with Gobuster requests.
- Added `--exclude-length` for filtering common wildcard response bodies by size.
- Store discovered paths relative to the URL root, including any base path supplied with `--url`.
- Added color-coded HTTP status codes to site-map output when writing to a terminal.

## Changed

- Site-map output disables ANSI colors for redirected or non-terminal output and when `NO_COLOR` is set.
