# xgobuster 1.2.0

`xgobuster 1.2.0` improves target web-service selection when multiple HTTP services are available.

## Added

- Added `--service <number>` to select a web service by its displayed number.
- Added Bash and Zsh completion for web-service numbers, including service descriptions in Zsh.

## Changed

- Keep interactive service selection when `--service` is not specified.
- Reject invalid service numbers and prevent `--service` from being combined with `--url`.
