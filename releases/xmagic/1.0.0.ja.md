# xmagic 1.0.0

`xmagic 1.0.0`は、元ファイルを変更せずにマジックナンバーを偽装したコピーを作成する初回リリースです。

## 追加

- 対応する形式とマジックナンバーを表示する`xmagic ls`を追加しました。引数なしの`xmagic`も同じ動作です。
- `xmagic set <type> <file>`を追加しました。既知形式では既存のマジックナンバーを置換し、未知形式では対象マジックナンバーを先頭へ挿入します。
- GIF、JPEG、PNG、PDF、ZIPに対応しました。
- 入力の最終拡張子を維持した出力名を自動生成し、同名ファイルが存在する場合は番号を付けます。
- 元ファイルを変更せず、入力と同じパーミッションで出力を作成します。
- 将来の修復機能で利用できるよう、操作方式、置換前後のバイト列、SHA-256をローカル状態へ保存します。
- BashおよびZshで、サブコマンド、形式、入力ファイルを補完します。

## 使用例

```sh
xmagic
xmagic set jpg image.png
xmagic set gif shell.php
```

`image.png`からは`image.jpg.png`、`shell.php`からは`shell.gif.php`を生成します。

マジックナンバーの変更はファイル形式の変換ではありません。生成物が対象形式として完全に解析できることは保証しません。
