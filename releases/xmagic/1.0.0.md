# xmagic 1.0.0

`xmagic 1.0.0` is the first release of a utility that creates a copy with a spoofed magic number without modifying the source file.

## Added

- Added `xmagic ls` to list supported types and magic numbers. Running `xmagic` without arguments behaves the same way.
- Added `xmagic set <type> <file>`. It replaces a recognized source magic number and prepends the target magic number when the source type is unknown.
- Added GIF, JPEG, PNG, PDF, and ZIP signatures.
- Added automatic output naming that preserves the final input extension and numbers collisions.
- Preserved the source file and its permission bits in generated output.
- Stored the operation mode, changed bytes, and SHA-256 values as local state for a future restoration command.
- Added Bash and Zsh completion for subcommands, types, and input files.

## Examples

```sh
xmagic
xmagic set jpg image.png
xmagic set gif shell.php
```

The examples create `image.jpg.png` and `shell.gif.php`, respectively.

Changing a magic number does not convert the underlying file format, and the generated output is not guaranteed to parse as a complete file of the target type.
