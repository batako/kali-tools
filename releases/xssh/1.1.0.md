# xssh 1.1.0

`xssh 1.1.0` adds local SSH public-key registration assistance.

## Features

- Adds `xssh key` to prepare an SSH public-key registration command.
- Generates `~/.ssh/id_ed25519` when `~/.ssh/id_ed25519.pub` is not available.
- Prints a copy-and-run command for adding the public key to the target user's `authorized_keys`.
- Creates the target `.ssh` directory and applies recommended permissions.
- Avoids registering the same public key more than once.
- Does not modify the target automatically; the printed command must be run on the target.

## Usage

```sh
xssh key
```
