# xssh 1.1.0

`xssh 1.1.0` に、ローカルの SSH 公開鍵をターゲットへ登録するための支援機能を追加しました。

## 機能

- SSH 公開鍵登録用のコマンドを準備する `xssh key` を追加しました。
- `~/.ssh/id_ed25519.pub` がない場合は、`~/.ssh/id_ed25519` を生成します。
- ターゲットの `authorized_keys` へ公開鍵を登録するコマンドを表示します。
- ターゲット側の `.ssh` ディレクトリ作成と推奨権限設定に対応します。
- 同じ公開鍵が登録済みの場合は重複登録しません。
- ターゲットは自動変更せず、表示されたコマンドをターゲット上で実行します。

## 使い方

```sh
xssh key
```
