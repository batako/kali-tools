# xffuf 1.0.0

## Added

- Added HTTP virtual host enumeration using ffuf.
- Added automatic selection of the ctx target, HTTP service, domain, and wordlists.
- Added response calibration against random hostnames and automatic filter suggestions.
- Added `--suggest` to display calibration statistics without recording trial data.
- Added a prompt after `--suggest` to run the proposed filter as a non-persistent trial.
- Added `--trial` to test filters against a real wordlist without recording logs, progress, or hosts.
- Added `--status`, `--next`, `--force`, and `--clear-cache` for wordlist progress management.
- Only stable normal runs update command logs, the wordlist cache, and `xhost`.
