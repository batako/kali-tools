# xffuf 1.1.0

## 追加

- query parameter名または値を探索する`xffuf param`を追加しました。
- `FUZZ`の位置とparameter名から用途を判定し、`wordlists`パッケージの適切なワードリストを自動選択します。
- parameter探索結果を`xweb`で閲覧できる構造でctxへ保存します。

## 変更

- 必須パッケージを`seclists`から`wordlists`へ変更しました。
- 既存のvhost探索も`/usr/share/wordlists`を基準にワードリストを選択します。
- Bash/Zsh補完をmode別にし、`vhost`と`param`で利用できるオプションだけを表示します。
- `-mc`、`-ml`、`-mr`、`-ms`、`-mw`のmatcher補完を追加し、parameter探索でmatcherを明示した場合は自動キャリブレーションを追加しません。

## 修正

- `/usr/share/wordlists/seclists`がsymlinkの場合にparameter用ワードリストを発見できない問題を修正しました。
