# xffuf 1.1.0

## Added

- Added `xffuf param` for query parameter name and value fuzzing.
- Added automatic wordlist selection based on the `FUZZ` position and parameter name.
- Saved structured parameter discoveries for viewing with `xweb`.

## Changed

- Replaced the required `seclists` dependency with `wordlists`.
- Existing vhost discovery now selects lists through `/usr/share/wordlists`.
- Made Bash and Zsh completion mode-aware so `vhost` and `param` only show valid options.
- Added completion for the `-mc`, `-ml`, `-mr`, `-ms`, and `-mw` matchers, and stopped adding automatic calibration when a matcher is explicitly selected for parameter discovery.

## Fixed

- Fixed parameter wordlist discovery when `/usr/share/wordlists/seclists` is a symlink.
