# xffuf 1.0.0

## 追加

- ffufを使ったHTTP Virtual Host探索を追加しました。
- ctxのターゲット、サービス、ドメイン、ワードリストを自動利用できるようにしました。
- ランダムなホスト名を使ったレスポンスキャリブレーションと自動フィルター提案を追加しました。
- `--suggest`で試行用の統計を表示できるようにしました。
- `--suggest`の提案後に、保存しない試走をそのまま実行できる確認プロンプトを追加しました。
- `--trial`でフィルターを実ワードリストへ適用し、ログ、進捗、ホストを保存せずに試せるようにしました。
- `--status`、`--next`、`--force`、`--clear-cache`でワードリストの進捗を管理できるようにしました。
- 安定した本実行の結果だけをコマンドログ、キャッシュ、`xhost`へ反映します。
