# req 0.1.0

req 0.1.0 は、`.req` ファイルに保存した生HTTPリクエストを再送するための小さなCLIツールです。

proxy、editor、noteなどから保存したHTTP requestを、terminalからもう一度送信したい場合に使えます。

## ハイライト

- `.req` ファイルから生HTTP requestを送信できます。
- request headerとbodyを保持して送信できます。
- `GET`、`POST`、`PUT`、`DELETE`、`PATCH`、`OPTIONS`、`HEAD` などの一般的なHTTP methodに対応します。
- 絶対URLまたは `Host` headerから送信先URLを解決します。
- 可能な場合、request headerからHTTP/HTTPSを推定します。
- `-S` または `--https` でHTTPSを明示できます。
- HTTP responseを標準出力へ表示します。

## 使い方

```bash
req request.req
req --https request.req
```

## 補足

- redirectは自動追従しません。
- `req` は独立したDebian packageとして配布されます。
