# req 1.0.0

`req 1.0.0` adds TLS validation controls and package installation support.

## Features

- Adds `-k`/`--no-tls-validation` to skip TLS certificate validation for test targets.
- Adds `--tls-verify` to explicitly keep TLS certificate validation enabled.

## Usage

```sh
req request.req
req -k request.req
```

`req` validates TLS certificates by default. Use `-k` only for test targets with expired or self-signed certificates.
