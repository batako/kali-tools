# xscp 1.0.0

`xscp 1.0.0` transfers files between the current ctx target and the local machine over SSH.

## Added

- Upload local files with `xscp upload <local> <remote>`.
- Download remote files with `xscp download <remote> <local>`.
- Omit the destination to use the source filename in the remote home directory for uploads or the current directory for downloads.
- Reuse ctx SSH credentials and discovered SSH service ports.
- Select a credential by ID or username, with the last used credential selected by default.
- Save successful and failed transfers to ctx command logs.
- Provide Bash and Zsh completion for transfer actions and paths.
