# xscp 1.0.0

`xscp 1.0.0` は、現在のctxターゲットとローカルマシンの間でSSH経由のファイル転送を行います。

## 追加

- `xscp upload <local> <remote>` でローカルファイルをアップロードできます。
- `xscp download <remote> <local>` でリモートファイルをダウンロードできます。
- 宛先を省略すると、uploadはリモートのホーム、downloadはカレントディレクトリにファイル名を引き継ぎます。
- ctxのSSH credentialと検出済みSSHサービスのポートを利用します。
- credentialをIDまたはユーザ名で指定でき、最後に使用したcredentialを既定選択します。
- 転送の成功・失敗をctxのコマンドログへ保存します。
- 転送操作とパスのBash・Zsh補完に対応します。
