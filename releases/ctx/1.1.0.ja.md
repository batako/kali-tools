# ctx 1.1.0

ctx 1.1.0 は、DBの自動マイグレーションを追加し、アップグレード時の安全性を高めるリリースです。

普段使うコマンドの動作を大きく変えるリリースではありません。今後ctxのDBが拡張されても、既存workspaceを安全に使い続けられるようにするための更新です。

## 新機能

- DBスキーマ変更が必要になった場合、ctxが自動でDBを更新できるようになりました。
- ctx 1.0.0で作成された既存DBを認識し、今後のmigrationに備えられるようになりました。
- 新規DBは最新の対応スキーマで作成されます。
- 実際のctx 1.0.0 DB fixtureを使ったmigrationテストをrelease checkに追加しました。

## 利用者向けの注意

- 手動でmigrationコマンドを実行する必要はありません。
- 既存workspaceは、アップグレード後もそのまま開ける想定です。
- ctxが未知または破損したDBスキーマを検出した場合、推測で処理を続けず停止します。

## 補足

- このリリースは主に互換性と信頼性のための更新です。
- CLIの見た目や通常操作はctx 1.0.0から変えない方針です。
