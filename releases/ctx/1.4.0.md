# ctx 1.4.0

`ctx 1.4.0` adds an optional short command for service management.

## Added

- Added the `sv` shortcut for `ctx service` when shell integration is enabled with `ctx init-shell --extra-shortcuts`.
- Added Bash and Zsh completion support for the `sv` shortcut.
- Updated the installation message to include `sv` among the optional shortcuts.
- Preserve all investigation data associated with a target when `xtarget set` changes its IPv4 address.
- Reuse services, hosts, credentials, command logs, scan history, web discoveries, and compatible `xgobuster` wordlist state after a target IP change.
- Added the `password.max-requests` configuration with a default of `10000` for password and username wordlist searches.
- Added automatic username wordlist discovery with quick, standard, and deep profiles.
- Show configuration defaults as a `DEFAULT` column in `xconfig ls`.
- Added password request configuration completion and help text.
