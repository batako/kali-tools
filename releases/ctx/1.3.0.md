# ctx 1.3.0

ctx 1.3.0 adds the data foundation for web content discovery integrations.

## Added

- Added the `web_discoveries` database table and schema migration.
- Added APIs for appending and listing web discovery observations.
- Added ordered `seclists` and `wordlists` providers for automatic wordlist selection.
- Added web wordlist run history for resumable discovery searches.
- Added a configurable line budget for automatic wordlist escalation.
- Preserved command log references for discovery executions.

This release does not add the `xgobuster` command yet. The Gobuster integration will be implemented separately on top of this foundation.
