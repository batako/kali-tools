# ctx 1.0.0

ctx 1.0.0 is a CLI tool for keeping target-focused work organized.

ctx helps you keep the current workspace, target, hostnames, scan results, credentials, notes, and command history together in one place.

## Highlights

- Create and manage workspaces for each investigation or task.
- Track a primary target IP and named targets.
- Save hostnames and manage ctx-owned `/etc/hosts` entries.
- Run commands with `ctx x` and save stdout, stderr, exit status, and timestamps.
- Add notes and review a combined timeline with `ctx log`.
- Run nmap scans with `ctx scan` and view saved service information with `ctx service`.
- Store credentials with scopes such as `ssh` or `wordpress`.
- Print shell prompt data with `ctx prompt`.
- Use JSON output for prompt, credential, service, and format information.
- Install shell completions and helper commands with `ctx init-shell`.

## Common Commands

```bash
ctx workspace init
ctx target set 10.10.10.10
ctx host add target.local
ctx scan -p 22,80,443
ctx service ls
ctx credential set ssh root toor
ctx note "interesting finding"
ctx log
ctx prompt
ctx x whoami
```

## Shell Integration

After running `ctx init-shell`, helper commands such as `xinit`, `xtarget`, `xscan`, `xservice`, `xcredential`, `xnote`, `xlog`, `xprompt`, and `x` are available in your shell.

Optional extra shortcuts can also provide `pj`, `ta`, and `cr`.

## JSON Output

ctx 1.0.0 includes versioned JSON output for external tools and add-ons.

Available JSON outputs:

- `formats`
- `prompt`
- `credential`
- `service`

## Maintenance

- `ctx doctor` checks your ctx setup and suggests fixes.
- `ctx reset` removes ctx-managed data and shell integration without uninstalling ctx.
- Debian packages are available for amd64 and arm64.
