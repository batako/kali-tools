# xdec 1.0.0

## 概要

文字列、ファイル、stdin を統一的に扱い、即時デコードと password recovery を一つのコマンドにまとめる Go 製フロントエンドを追加しました。John と hashcat は内部 backend として自動選択されます。

## 主な機能

- 文字列、`-f` ファイル、stdin の入力と候補抽出
- 実在する通常ファイルを位置引数で渡す省略記法
- Base64 / hex の即時デコード
- MD5、NTLM、MD4、SHA-1、SHA-256、bcrypt、Argon2 prefix の分類
- 高コスト解析前の確認と、非対話時の `--yes`
- ctx password wordlist の自動利用と複数 wordlist の継続実行
- wordlist ごとの xlog 子 step と xdec 親ログ
- SSH 秘密鍵の自動検出、暗号化状態の判定、`ssh2john` 経由の解析
- 未暗号化 SSH 鍵の即時判定（John を実行しない）
- ユーザ名の抽出と、scope が確定した復元 credential の ctx 自動保存
- 入力単位の state、完了済み解析のスキップ、保存済み結果の再表示
- `--refresh` による対象入力だけの再解析
- 通常出力と JSON 出力、bash / zsh 補完
- `--online-help` によるバージョン付きオンラインドキュメント URL の表示
- `decode`、`help`、`version` サブコマンドと、フラグ形式のヘルプ・バージョン表示
- 自動判定を上書きする `--file` / `--string` の明示的な入力型指定

## セキュリティ

復元結果は state に保存されるため、state ファイルは 0600 で管理します。秘密鍵の内容とパスフレーズは xlog に記録しません。
