# xhydra 1.1.0

## Changed

- Unified automatic password and username selection on ctx wordlist recommendations.
- Removed independent filesystem discovery and hard-coded password candidates.
- Preserved explicit `--password-list` and `--user-list`, searched-word tracking, and request limits.
- Repeating the same command now resumes from unseen words in ctx recommendation order.
- Set Hydra's SSH task count to the recommended default of four to respect common SSH connection limits.
- Added `-t, --tasks <number>` to override the SSH task count.
- Limited FTP to four parallel tasks by default and allowed overriding it with `--tasks`.

## Removed

- Removed `--force`; use `--clear-cache` to restart scoped progress.
