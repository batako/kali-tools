# xhydra 1.0.0

`xhydra 1.0.0` は、HTTPフォーム認証にHydraを実行し、成功したcredentialをctxへ保存します。

## 追加

- `-r`/`--request` で生HTTPリクエストファイルを入力できます。
- `--url` と `--data` でリクエストファイルを使わずに実行できます。
- 指定したユーザー名・パスワードのフォーム項目をHydra用に置換します。
- `--fail-json <field=value>` でJSONレスポンスの失敗条件を指定できます。
- インストール済みのパスワードワードリストを自動選択し、`-P`/`--password-list` で明示指定もできます。
- 成功したHTTP credentialをctxへ保存します。
- 成功・失敗に関係なく、コマンド出力と実行状態をctxログへ保存します。
