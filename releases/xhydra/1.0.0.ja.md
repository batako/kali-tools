# xhydra 1.0.0

`xhydra 1.0.0` は、HTTP・SSH・FTP・SMBの認証にHydraを実行し、成功したcredentialをctxへ保存します。

## 追加

- `-r`/`--request` で生HTTPリクエストファイルを入力できます。
- `--url` と `--data` でリクエストファイルを使わずに実行できます。
- 指定したユーザー名・パスワードのフォーム項目をHydra用に置換します。
- `--fail-json <field=value>` でJSONレスポンスの失敗条件を指定できます。
- SSH・FTP・SMBモードに対応し、サービスを自動選択できます。`--service`、`--host`、`--port` で明示指定もできます。
- インストール済みのパスワードワードリストを自動選択し、`-P`/`--password-list` で明示指定もできます。
- 重複および検索済みパスワードを除外し、設定した試行上限までワードリストを段階的に実行します。
- `--password` で固定パスワードによるユーザ名検索に対応し、ユーザ名ワードリストを自動選択できます。`-L`/`--user-list` で明示指定もできます。
- ユーザ名検索の進捗を、対象・サービス・固定パスワードごとに分離して管理します。
- `--status` でワードリスト単位の進捗を確認でき、`--force` で自動検索を再実行し、`--clear-cache` で対象範囲の進捗を削除できます。
- 成功したHTTP・SSH・FTP・SMBのcredentialをctxへ保存します。
- パスワードワードリストごとにコマンドログを保存し、成功・失敗に関係なく出力と実行状態を記録します。
- モードとオプションのBash・Zsh補完に対応します。
